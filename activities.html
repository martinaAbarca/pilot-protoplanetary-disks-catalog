Programming activities with disk data / Actividades de programación con datos de discos

This section presents introductory programming activities based on real protoplanetary disk data. The aim is to help students connect scientific interpretation with practical work using astronomical images.

Esta sección presenta actividades introductorias de programación basadas en datos reales de discos protoplanetarios. El objetivo es ayudar a los estudiantes a conectar la interpretación científica con el trabajo práctico usando imágenes astronómicas.

The activities focus on FITS images, disk geometry, and simple analysis tools that make it possible to identify rings, gaps, and other disk substructures.

Las actividades se centran en imágenes FITS, geometría de discos y herramientas simples de análisis que permiten identificar anillos, brechas y otras subestructuras del disco.

Activity 1 / Actividad 1

English

Radial profile from a FITS image

In this activity, students work with a real FITS image of the protoplanetary disk HD 163296 and compute a radial brightness profile. The goal is to explore how the intensity changes with distance from the central star and to relate those variations to rings, gaps, and disk substructures visible in the image.

Suggested materials - FITS image of HD 163296 - Jupyter notebook or Python script - Short guide with the required libraries - Scientific context from the catalog entry

What students practice - Reading FITS files - Visualizing astronomical images - Understanding disk geometry - Measuring intensity as a function of radius - Identifying rings and gaps from real data

Files - FITS image - Complete package (.zip)

Recommendation - Download the complete package so the notebook and the FITS image remain in the same folder.

Español

Perfil radial a partir de una imagen FITS

En esta actividad, los estudiantes trabajan con una imagen FITS real del disco protoplanetario HD 163296 y calculan un perfil radial de brillo. El objetivo es explorar cómo cambia la intensidad con la distancia a la estrella central y relacionar esas variaciones con anillos, brechas y otras subestructuras visibles en la imagen.

Materiales sugeridos - Imagen FITS de HD 163296 - Notebook o script en Python - Guía breve con las librerías necesarias - Contexto científico desde la ficha del catálogo

Qué se practica - Lectura de archivos FITS - Visualización de imágenes astronómicas - Comprensión de la geometría del disco - Medición de intensidad en función del radio - Identificación de anillos y brechas a partir de datos reales

Archivos - Imagen FITS - Paquete completo (.zip)

Recomendación - Se recomienda descargar el paquete completo para que el notebook y la imagen FITS queden en la misma carpeta.

Activity 2 / Actividad 2

English

Deprojection and polar analysis of a disk image

In this activity, students use a disk image to apply a geometric deprojection and then transform it into polar coordinates. This makes it easier to identify rings, asymmetries, and localized structures in the disk.

Suggested materials - FITS image of an inclined disk - Notebook or Python script - Short methodological guide

What students practice - Using inclination and position angle - Deprojecting a disk image - Converting an image into polar coordinates - Identifying substructures more clearly

Español

Deproyección y análisis polar de una imagen de disco

En esta actividad, los estudiantes usan una imagen de un disco para aplicar una deproyección geométrica y luego transformarla a coordenadas polares. Esto facilita la identificación de anillos, asimetrías y estructuras localizadas en el disco.

Materiales sugeridos - Imagen FITS de un disco inclinado - Notebook o script en Python - Guía metodológica breve

Qué se practica - Uso de inclinación y ángulo de posición - Deproyección de una imagen de disco - Conversión de una imagen a coordenadas polares - Identificación más clara de subestructuras
